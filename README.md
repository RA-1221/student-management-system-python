# Student Management System

A simple Student Management System built using Python and Object-Oriented Programming (OOP).

## 📌 Features
- Add Student
- View All Students
- Search Student by ID
- Delete Student

## 🧠 Concepts Used
- Classes and Objects
- Constructors (__init__)
- Methods
- Lists
- Loops (while, for)
- Conditionals (if/elif)

## 🚀 How it works
The program runs in a menu-based loop where the user can choose different operations to manage student data.

## 💡 Purpose
This project was built for learning OOP concepts and practicing Python programming fundamentals
